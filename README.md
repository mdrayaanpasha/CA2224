# Object Oriented Programming With Java - SJU Archive

This is a digital archive of code taught in classes of **Object Oriented Programming With Java @ SJU 2nd SEM class of 2027** during the academic year **2024-25**.

## Repository Structure

The repository is organized into different labs, assignments, and class exercises:

### Assignments
- **[Assignment 1](./assignment1/)**: Basic Java programs like prime numbers, fibonacci, matrix operations, etc.
  - **[Lab 2](./assignment1/lab2/)**: Month conversion, rectangle area, factorials, and number comparisons.
- **[Assignment 2](./assignment2/)**: Advanced topics including Abstract Classes, Interfaces, Exception Handling, and GUI (Swing).
  - **[School Project](./assignment2/School/)**: Implementation of Teachers and Students classes.

### Lab Exercises
- **[Lab 2](./lab2/)**: Method Overloading.
- **[Lab 3](./lab3/)**: Abstraction, Prime numbers, Circle calculations, and Student management.
- **[Lab 4](./lab4/)**: Exception handling (Negative expressions, String length).

### Class Work & Miscellaneous
- **[Class 1](./class1/)**: Static members and Bank account simulation.
- **[File Handling](./fileHandling/)**: File creation, reading, and writing operations.
- **[GUI](./GUI/)**: Java Swing/AWT lecture examples.
- **[years.java](./years.java)**: Root level exercise.

## How to Run
To compile and run any Java file, use the following commands in your terminal:

```bash
javac FileName.java
java FileName
```
